# trygame16
Ты - путешественник, пробирающийся в подземелье в поисках приключений и несметных богатств. На пути тебе встретятся опасные враги и ценные сокровища. Удачи!
## **Особенности**
- Интересные механики
- 6 уровней и финальный босс
- Звуковое сопровождение и незаьываемая музыка
- Можно играть без мыши - только клавиатура!
##**Как играть**
### Управление
- WASD - перемещение
- ←→↑↓ - атака, перемещение в сундуке, в меню паузы - управление громкостью звука(↑↓) и музыки(←→)
- SPACE - использовать(открыть дверь, сундук; использовать что-то в руках)
- Tab - пауза
- QE - управление инвентарем
- ctrl - спринт
- ENTER - выйти из сундука
### Цель игры - пройти все уровни и одолеть босса
##Установка
- Скачать архив последней версии, распаковать
- Перейти в Releases(https://github.com/NikakSIA/trygame16/releases/tag/gamefile) и скачать trygame.exe
- Закинуть .ехе файл в директорию trygame16 в распакованном архиве
- Создать ярлык .ехе файла и запускать из удобного места
